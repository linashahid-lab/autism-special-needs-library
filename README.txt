HOW TO OPEN
1. Right-click the zip and choose Extract All.
2. Open the folder.
3. Double-click index.html.

HOW TO PUT ONLINE FOR FREE WITH GITHUB PAGES
1. Create a free GitHub account.
2. Create a new repository.
3. Upload everything from this folder, including the assets, essays and originals folders.
4. Go to Settings > Pages.
5. Choose Deploy from branch, main branch, root folder.
6. GitHub will give you a public website link.

WHAT IS INCLUDED
- A colourful homepage.
- An About page.
- Four essay pages.
- Original PDF buttons on each essay page.
- Original PDF files in the originals folder.

IMPORTANT BEFORE PUBLIC
Check every PDF and page for private details, student numbers, lecturer comments, private case information, school names, or anything you do not want online.
